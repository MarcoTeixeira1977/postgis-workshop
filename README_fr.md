PostGIS workshop
================

Un workshop d'initiation à POSTGIS pour découvrir quelques unes de ses fonctionnalités.

Il utilise :

* PostGIS 2.x
* QGIS et son DBManager
* PgAdmin3 (ou 4)

Suivez les étapes fournies dans le répertoire _doc_/_fr_.
